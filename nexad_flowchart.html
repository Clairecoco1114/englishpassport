1
框架一：外部赛马 — 明确「赢」的逻辑
战略层
①
明确目标与业务理解
深入理解客户业务，梳理增长目标与痛点
②
定义正确的 KPI （核心）
与客户对齐成功标准，确保 KPI 可量化、可追踪
③
制定获胜策略
渠道组合、测试策略、资源分配节奏
④
高效执行赛马
更快测试、快速迭代优化，按 KPI 客观评判
⑤
放量与持续优化
找到可复制最优路径，沉淀方法论与数据资产
⚠ 赛马失败的两大原因
✕ KPI 对齐错误 → 数据好看但无实际价值，无法放量
✕ 测试预算/时间不足 → 还没跑出方向，对手已领先
核心结论：KPI 必须在 Onboarding 阶段对齐。但「选哪个 KPI」取决于客户当前所处的业务阶段 → 需要先诊断
先诊断，再选KPI
2
框架二（入口）：产品诊断 — 客户现在处于哪个阶段？
诊断层
诊断视角：客户跨所有渠道的整体状态。 关键指标 = 关键行为 CPA 达标 或 ROAS 为正，付费渠道跑出任意一个即算。
甲方是否已决定要做付费投放？
尚未决定 / 犹豫中
常见场景：自然流量数据好，不确定是否要做付费
甲方的顾虑
·自然流量用户质量这么好，付费用户会不会差很多？
·现在自然数据很好，付费真的有必要吗？
Agency 需要帮甲方建立的认知
①自然流量和付费流量是两个独立的增长引擎，数据不能互相套用，付费用户质量确实会不同
②自然流量再好都有天花板，想要规模化增长，付费是必经路径
③付费能否跑正，跟自然数据好不好关系不大，跟产品本身的 ROI 逻辑关系更大。只要产品逻辑成立，付费大概率能为正，只是成本不同
Agency 的介入价值
帮甲方做这个决策，本身就是 Onboarding 阶段的核心工作之一。决策清晰后，再进入右侧诊断流程。
认知对齐后 → 决定做付费 →
已决定做付费
付费渠道是否已跑出关键指标？
需同时满足两个条件：
① 指标达标 关键行为CPA / ROAS 符合目标
② 有一定规模 数据有统计意义，加预算后指标仍能保持稳定
是（指标达标 + 有规模）
优先接该渠道
已有数据基准，KPI 直接对标。量级不够再考虑开新渠道（KPI需重新协商）
⚠ 执行从 CPA 起步 → 过渡到关键行为CPA / ROAS（平台学习期，非验证产品）
否（含指标达标但规模太小）
回到 0→1 逻辑处理
规模太小 = 数据无统计意义，加预算后指标是否稳定未知，不能直接对标放量
先决策渠道 → 有关键行为数据：目标关键行为CPA，从注册起步
无任何数据：目标注册CPA，与其他Agency同起跑线
诊断结论输出：确定 Agency 的渠道切入方向、合作 KPI 目标与执行起跑层级 → 交给框架三匹配具体指标
前置 甲方犹豫是否做付费 → Agency 先做认知对齐：自然流量有天花板，付费是独立引擎，产品ROI逻辑成立则大概率能跑正
A 付费渠道指标达标 且有一定规模 → 优先接该渠道，KPI直接对标；量级天花板再开新渠道。执行从CPA起步 （平台学习期）
注意 指标达标但规模太小 → 数据无统计意义，回到B线处理，不能直接对标放量
B1 完全0→1，有关键行为数据 → 先决策渠道，目标关键行为CPA，从注册起步验证付费流量质量
B2 完全0→1，无任何数据 → 先决策渠道，目标注册CPA，所有Agency同起跑线，竞争在执行速度
根据阶段，匹配KPI维度
3
框架三：KPI 选择 — 不同阶段用什么指标？
执行选择层
← 测试/验证阶段（0→1）
放量/增长阶段（1→N）→
上层漏斗
获客类 KPI
CPA-注册
产品初期，用户价值未验证，需扩大用户基数
中层漏斗
行为类 KPI
CPA-关键行为
核心功能已定义，验证用户活跃度与转化质量
下层漏斗
效果类 KPI
CPA-付费
已验证商业模式，关注付费转化效率
收益层
收益类 KPI
ROAS / LTV / ROI
规模化投放阶段，关注整体回报与盈利
选定 KPI 维度后，设定三值：Target（目标值）/ Test Range（测试区间）/ Cut-off（止损线） → 进入执行判断
KPI已定，进入执行
4
框架二（主体）：执行判断 — 测试、收敛、放量、升级
执行判断层
①
学习期（3-5天）
收集初始数据，排除极差方向。不做最终判断，目标是缩小测试范围。
②
多方向测试 — 数据对比，按 KPI 统一判断
素材 / 受众 / 出价多方向并行，直到能判断转化趋势。主 KPI 成本 + 下游转化率双维度评估。
素材方向 受众方向 出价策略 渠道组合
③
结果判断：对比 Target / Test Range / Cut-off
≤ Target → 放量 在 Test Range → 继续优化 > Cut-off → 停止该方向
若所有方向均超 Cut-off 且无改善趋势（7-10天），触发产品诊断流程（见风险处理）。
④
找到赢家路径 → 进入 Scaling
扩展性假设成立（加预算 / 拓人群 / 拓素材后效果稳定）后，进入规模化放量阶段。
⑤
放量后是否稳定？
放量后持续监控：成本是否维持在 Target 区间，下游转化率是否跟随。
成本稳，下游转化率下降 → 持续优化素材/落地页 成本高 → 回退调整
⑥
进入增长阶段 — 继续当前 KPI / 评估升级
满足 KPI 升级条件（上层稳定 + 下层转化成立）时，升级 KPI 维度。
⑦
KPI 升级 → 回到 Step 2 重新诊断
注册CPA → 关键行为CPA → 付费CPA → ROAS，每次升级后回到框架二入口重新诊断阶段，进入下一轮循环。
← 循环回到框架三，重新匹配下一阶段 KPI
✓ 稳定放量
持续增长 / 长期合作
定期复盘，KPI 逐步升级，向 ROAS 阶段推进
△ 放量失效
ROI 下降 / 成本上升
回退重新测试；检查素材疲劳、竞争变化、产品问题
✕ 方向不成立
触发产品诊断
识别是「投放问题」还是「产品/商业模式问题」，三叉路口决策
